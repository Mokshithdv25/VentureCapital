Add your dashboard screenshots here.

## Image Guidelines:

- **Format**: PNG or JPG
- **Size**: Recommended 1920x1080 or similar
- **Naming**: Use kebab-case (e.g., `dashboard-overview.png`)
- **File Size**: Keep under 2MB per image for faster loading

## How to Take Screenshots:

1. Open your live app: https://venturecapital-ujqvnjxgynh5y9ckpesqzi.streamlit.app
2. Navigate to each tab
3. Take a screenshot (Cmd+Shift+4 on Mac, Snipping Tool on Windows)
4. Save to this `screenshots/` folder
5. Update the image paths in the main README.md
